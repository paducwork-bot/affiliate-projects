# Content

Blog posts, social content, email sequences for affiliate marketing.

## Categories

- `blog/` - Long-form articles
- `social/` - LinkedIn, Twitter, Reddit posts
- `email/` - Email sequences
- `video/` - Video scripts

## Content Workflow

1. Research program → `../programs/`
2. Write content → `./blog/` or `./social/`
3. Build landing page → `../landing-pages/`
4. Track performance → `../analytics/`
