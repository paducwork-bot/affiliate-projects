# Affiliate Projects

Workspace for affiliate marketing projects using [affiliate-skills](https://github.com/Affitor/affiliate-skills).

## Structure

```
affiliate-projects/
├── programs/       # Program research & comparison
├── research/       # Market research, competitor analysis
├── content/        # Blog posts, social content
├── landing-pages/  # Landing page HTML/CSS
├── analytics/      # Performance tracking, A/B tests
└── notes/          # Quick notes, ideas
```

## Quick Commands

```bash
# Search programs
affiliate-check search "AI tools" --recurring --limit 10

# Get top programs
affiliate-check top --sort trending

# Program details
affiliate-check info heygen

# Compare programs
affiliate-check compare heygen synthesia copy-ai
```

## Tools

- **affiliate-check** - CLI for Affitor API
- **affiliate-skills** - 32 skills for affiliate marketing
